# HR Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **HR Analytics Dashboard** developed using **Microsoft Power BI**. The dashboard helps HR professionals analyze employee attrition trends, workforce demographics, job satisfaction levels, and salary-related insights.

The goal of this project is to transform raw HR data into meaningful visualizations that support data-driven decision-making and improve employee retention strategies.

---

## 🎯 Objectives

- Analyze employee attrition across different departments and demographics.
- Identify workforce trends based on age groups, salary ranges, and education fields.
- Monitor key HR metrics through KPI cards.
- Visualize employee satisfaction and attrition patterns.
- Build an interactive dashboard using Power BI.

## 🧹 Data Preparation

The following preprocessing steps were performed:

- Imported HR dataset into Power BI.
- Verified and corrected data types.
- Checked for missing values and duplicates.
- Removed duplicate employee records.
- Renamed fields for better readability.
- Created measures using DAX for KPI calculations.

---

## 📊 Dashboard KPIs

### Total Employees
Displays the total number of employees in the organization.

### Attrition Rate
Measures the percentage of employees who left the organization.

**Formula:**

```text
Attrition Rate = (Employees Left / Total Employees) × 100
```

### Average Monthly Income
Displays the average monthly income of employees.

## 📈 Visualizations Included

### 1. Donut Chart
**Attrition by Education Field**

Shows the distribution of employee attrition across different education backgrounds.

### 2. Clustered Column Chart
**Attrition by Age Group**

Analyzes attrition trends among different age categories.

### 3. Horizontal Bar Chart
**Attrition by Salary Slab**

Displays employee attrition based on salary ranges.

### 4. Matrix Visual
**Job Role vs Job Satisfaction Level**

Provides insights into employee satisfaction levels across various job roles.

### 5. Line Chart
**Employees Left by Years at Company**

Shows attrition trends based on employee tenure.

### 6. Column Chart
**Employees Left by Department**

Identifies departments with the highest employee attrition.

### 7. Interactive Slicers

- Department
- Gender

Allows dynamic filtering across all visuals.

---

## 🔍 Key Insights

- Attrition rate is approximately **16.1%**.
- Employees in the **26–35 age group** show the highest attrition.
- Employees earning **up to 5K salary slab** have the highest turnover.
- The **Research & Development** department records the highest attrition.
- Attrition decreases as employee tenure increases.
- Job satisfaction levels vary significantly across different job roles.
